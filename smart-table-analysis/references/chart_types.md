# 图表类型选择指南

## 分析意图 → 图表类型映射

根据用户的分析意图，选择最合适的图表类型。这是图表生成的核心决策参考。

### 映射表

| 分析意图 | 推荐图表 | 适用场景 | chart_generator 参数 |
|---------|---------|---------|---------------------|
| **比较** 各组数据大小 | 柱状图 (bar) | 比较不同地区/产品/月份的销售额 | `--chart-type bar` |
| **排名** TOP N | 水平柱状图 (bar horizontal) | 排名、排行榜 | `--chart-type bar --orientation horizontal --top-n N` |
| **趋势** 随时间变化 | 折线图 (line) | 月度趋势、同比走势 | `--chart-type line --add-ma True` |
| **分布** 数据分散情况 | 直方图 (histogram) | 年龄分布、金额分布 | `--chart-type histogram --bins 30` |
| **构成** 各部分占比 | 饼图 (pie) | 地区占比、品类占比 | `--chart-type pie --top-n 8` |
| **关系** 两个变量的关联 | 散点图 (scatter) | 价格vs销量、温度vs湿度 | `--chart-type scatter` |
| **相关** 多变量相关矩阵 | 热力图 (heatmap) | 相关系数矩阵 | `--chart-type heatmap --corr-data JSON` |
| **异常值** 检测离群点 | 箱线图 (box) | 检测异常值、对比分布 | `--chart-type box` |
| **范围** 随时间的区间变化 | 面积图 (area) | 趋势+范围、累计值 | `--chart-type area` |
| **多指标对比** | 分组柱状图 (grouped_bar) | 多维度对比 | `--chart-type grouped_bar` |

### 选择决策树

```
用户要看什么？
├─ 变化过程 → 有时间轴吗？
│  ├─ 是 → 折线图 (加移动平均)
│  └─ 否 → 对比前后 → 柱状图
├─ 占多少 → 饼图 (超过8类用柱状图代替)
├─ 聚在一起还是散开 → 直方图/箱线图
├─ 两个东西有没有关系 → 散点图
├─ 多个东西互相有没有关系 → 热力图
├─ 排名 → 水平柱状图
└─ 多个指标一起对比 → 分组柱状图
```

### 图表配置最佳实践

1. **柱状图**：
   - 超过15个类别时，自动截取 TOP 15 + "其他"
   - 正负值时用红涨绿跌配色（中国惯例）
   - 排名场景使用水平柱状图

2. **折线图**：
   - 默认添加7日移动平均虚线
   - 多条线不超过5条，否则分图
   - 超过200个点时自动采样

3. **饼图**：
   - 超过8个类别时合并为"其他"
   - 显示百分比标签
   - 确保总和有意义

4. **散点图**：
   - 大数据量(>5000点)时设alpha=0.3避免重叠
   - 支持第三维度映射为颜色/大小

5. **热力图**：
   - 使用 RdYlGn 色谱（红=负相关，绿=正相关）
   - 在单元格内显示数值
   - 强相关(|r|>0.7)加粗标注

6. **箱线图**：
   - 分组不超过10个
   - 标注异常值点
   - 显示均值标记

### 图表标题命名规则

- 格式：`[分析维度] × [指标] — [分析类型]`
- 示例：`地区 × 销售额 — 对比分析`、`时间 × 订单量 — 趋势走势`
- 避免纯英文标题，用中文描述意图
